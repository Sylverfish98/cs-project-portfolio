### Projeto de Web Standards ###

Um simples site responsivo para uma escola de línguas fictícia feito em HTML e CSS vanilla, projeto da matéria de Web Standards.

O código-fonte se encontra na pasta ./public para permitir o deploy na Vercel sem frameworks ou código adicional.

Link para o site: https://cs-project-portfolio.vercel.app/

Link para o repositório do Github: https://github.com/Sylverfish98/cs-project-portfolio/
